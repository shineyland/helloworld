04 / 12

Authentication Flow

Token Strategy

Access Token

15 min lifetime • Stored in memory

Refresh Token

7 day lifetime • HTTP-only cookie

Login Flow

1. User enters credentials
2. Server validates & generates tokens
3. Access token sent in response
4. Refresh token set as cookie
5. Redirect to role dashboard
05 / 12

API Design

Authentication Endpoints

POST /api/v1/auth/login → Authenticate user POST /api/v1/auth/logout → Invalidate refresh token POST /api/v1/auth/refresh → Get new access token GET /api/v1/auth/me → Get current user [Auth] PUT /api/v1/auth/change-password
